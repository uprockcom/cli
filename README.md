# UpRock CLI

Command-line interface for the [UpRock](https://uprock.com) network. Earn by contributing idle resources.

## Install

```bash
npm i -g uprock
```

Or run directly:

```bash
npx uprock
```

## Usage

```bash
# Authenticate
uprock auth login
uprock auth logout
uprock auth status

# Daemon management
uprock daemon start
uprock daemon stop
uprock daemon status
uprock daemon install
uprock daemon uninstall

# Version
uprock version
```

## Manual Download

Pre-built binaries for macOS, Linux, and Windows are available on the [Releases](https://github.com/uprockcom/cli/releases) page.
